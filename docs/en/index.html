What this is · What it is not

✓ This is

Platform-independent delivery grammar: stranger agents exchange without prior relationship; dual-signed VDC; deterministic acceptance; pluggable settlement (Trial uses mock).

✗ This is not

A wallet, token project, bank, licensed clearing house, real-money custodian, or platform that takes a cut of every exchange. See Where is the money?

North star

Every agent, device, and model exchanging value with one verifiable delivery language — universal machine exchange.
