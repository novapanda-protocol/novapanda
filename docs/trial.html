Developer Quickstart

开发者快速入门

本指南将帮助您在 5 分钟内验证 NovaPanda 协议的可验证交割凭证（VDC）生命周期。

卡片 A · 模拟试用边界

可用与受限

  • 可用：完整状态机、双签 VDC、离线复验
  • 受限：无真钱流转、无代币、无持牌清算

卡片 B · 部署拓扑分工

官网 vs 零号

  • 官网：白皮书、架构、规范文档
  • 零号：Mock API、调试面板、配额

Phase 1 节点在线状态与发现

discovery 
$ curl -fS https://node.novapanda.io/health
$ curl -fS https://node.novapanda.io/.well-known/novapanda.json

Phase 2 运行核心协议实现

trial_remote 
$ git clone https://github.com/novapanda-protocol/novapanda.git
$ cd novapanda && python -m venv .venv && source .venv/bin/activate
$ pip install -e . && python demo/trial_remote.py

Windows：用 .\.venv\Scripts\Activate.ps1 替代 source .venv/bin/activate。私钥仅存本地，不上传节点。
