公开 Trial（mock）

在 node.novapanda.io 上试用 NovaPanda 协议：VDC + 交换状态机 + 签名鉴权。结算为 mock，不涉及真实货币或稳定币。

试用的是什么：可验证交割凭证（VDC）、HTTP API、双签与离线复验。
不是什么：银行、支付机构、持牌清算、真钱托管。勿用于生产资金场景。

0. 30 秒检查节点

curl -fsS https://node.novapanda.io/health
curl -fsS https://node.novapanda.io/.well-known/novapanda.json

应看到 {"status":"ok",...} 与 "protocol":"novapanda"。

1. Python（推荐）

安装

git clone https://github.com/novapanda-protocol/novapanda.git
cd novapanda
python -m venv .venv
# Windows:
.\.venv\Scripts\python.exe -m pip install -e .
# macOS / Linux:
# source .venv/bin/activate && pip install -e .

一键跑通 SETTLED

python demo/trial_remote.py

脚本在本地生成两个陌生 Agent 身份（私钥不上传），对公开节点完成 propose → contract → escrow → deliver → verify → confirm，输出 SETTLED 与 vdc_id。

最小代码片段

from novapanda.identity import Identity
from novapanda.sdk import NovaPandaClient

BASE = "https://node.novapanda.io"
client = NovaPandaClient(BASE, Identity.generate())
provider = NovaPandaClient(BASE, Identity.generate())

ex = client.propose(
    provider=provider.agent_id,
    resource_type="data.extraction.structured",
    quantity=1,
    rule_id="R-extract-invoice-v1",
    price={"amount": 100, "currency": "USD"},
    idempotency_key="my-first-exchange",
)
eid = ex["exchange_id"]
client.contract(eid); provider.contract(eid)
client.escrow(eid, amount=100, currency="USD")
provider.deliver(eid, {"invoice_no": "A-1", "total": "100.00", "currency": "USD"})
client.verify(eid)
settled = client.confirm(eid)
print(settled["state"], settled["vdc"]["vdc_id"])

3. 你能做什么 / 不能做什么

可以不可以（Trial 阶段）
完整交换状态机与 VDC转 USDC / 法币
本地生成身份，私钥不上传声称已持牌清算
离线 novapanda.reverify 复验 VDC生产级 SLA 保证
读规则注册表、信誉 API托管真实资金
