## 北极星（图必须传达的一句话）

**智能万物交换**  
一切**智能体**与**智能 IoT**（车、桩、机器人、无人机、传感与边缘设备……）——在**无预建关系、无需人参与**的前提下，用同一套可验证交割语言自主交换价值。

核心不是「人点按钮完成一笔交易」，而是：  
**机器对机器（A2A / Device-to-Device）自主 propose → 交付 → 双签 VDC**，人退到规则与终审之外。

成功标志：  
陌生智能体 / 智能设备能否互认一张可离线复验的 VDC——**全程可不经人工值守**。

---

## 必须钉死的叙事重点

1. **主体 = 所有智能体 + 智能 IoT**（对等，不是「App 很大、设备很小」）  
2. **无人参与也可完成交换**（无人值守、自动履约、自动验真）  
3. **同一套语法**贯穿软件 Agent 与物理设备  
4. **车 · 桩 · 电能 · 机器人 · 无人机 · 到场巡检闭环**是灯塔剪影，用来证明「万物」——不是行业海报堆砌

---

## 叙事结构（现在 → 灯塔，同一脊梁）

### 左 · 现在可排练（软件智能体）

陌生 **Agent ↔ Agent**，无人审批也能跑通：

| 剪影 | 交换什么 | 要点 |
|------|----------|------|
| 结构化抽取 | 数据/字段交付 | 自动验收 + 双签 VDC |
| 可验收任务 | 任务完成 | 规则判定，非口头确认 |
| Token / 算力 | 计量交割 | 黑盒账单 → 可复验凭证 |
| 嵌套尽调包 | 多腿软件交付 | 多张 VDC，替身排练物理多腿 |

角标文案：`无人值守 · 软件模拟舱`

### 右 · 灯塔（智能 IoT · 无人参与）

主体换成设备与机器人——**Owner 可不在线盯盘**，设备按协议自主交换：

| 主体 | 交换什么 | 无人参与含义 |
|------|----------|--------------|
| 车 ↔ 充电桩 | 直流电能（kWh） | 插枪/会话级自动 propose–交付–VDC |
| 机器人 ↔ 调度 Agent | 可证明任务完成 | 任务结束自动签收，无需人点确认 |
| 无人机 ↔ 场站 / Owner Agent | 巡检结果交付 | 航线完成即交割，结果凭证自动落袋 |
| 车 · 无人机 · 机器人 | **到场巡检闭环** | 多设备串行/并行自动对账，嵌套多张 VDC |

角标文案：`智能 IoT · 无需人参与亦可交换`

### 中轴 · 共用语法（整图灵魂）

```text
智能体 / 智能 IoT（对等身份）
  → 无预建关系 · Manifest 发现
  → 自主 propose / deliver（无人值守）
  → 双签 VDC（真理层）
  → 离线任何人可复验
  → 清算轨可插拔（下游，非主角）
```

构图要点：软件 Agent、车、桩、机器人、无人机画成**同一视觉权重的对等节点**，共同指向一枚 **VDC**——强调「万物同语」，不是某个人类控制台在中间转发。
