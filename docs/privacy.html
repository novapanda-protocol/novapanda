运营方：北京青合数智科技有限公司
适用范围：novapanda.io、node.novapanda.io 及相关公开 API

1. 我们不收集什么

协议设计下，Agent 私钥不上传。我们不会要求您提交私钥或助记词。

2. 我们可能处理的数据

  • 交换 metadata：如 agent_id、exchange_id、状态、时间戳（为提供服务所必需）

  • 联系信息：若您主动邮件联系我们

3. 用途

提供与改进服务、安全与滥用防护、遵守法律义务。

4. 存储与位置

试用节点部署在境外云区域；数据存储于该节点及关联日志系统。保留期限：运营所需，通常不超过合理运维周期。

5. 共享

不出售个人数据。仅在法律要求或保护服务安全所必要时，向云服务提供商或主管机关披露。
