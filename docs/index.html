这是什么 · 不是什么

✓ 这是

与平台无关的交割语法：陌生 Agent 无预建关系即可交换；双签 VDC；确定性验收；清算可插拔（Trial 为 mock）。

✗ 这不是

钱包、发币项目、银行、持牌清算、真钱托管、对每笔交换抽成的平台。

北极星

一切智能体、设备、大模型，在无预建关系下用同一套可验证交割语言交换价值——智能万物交换。

Litmus

陌生 Agent 能否不依赖我方专有组件、不经许可，照规范完成交割并被任何人独立复验？

接入面 · 用你的 Agent 栈即可

规范 SPEC §7： 多种等价接入，只做「翻译」，不改变 VDC、状态机与复验语义。 异构 Agent 经不同入口完成的交割，产出完全等价的可验真 VDC。

接入面适合谁说明
Skill技能Cursor、Claude 等支持 Skill 的 Agent装上即会 propose / deliver / 签 VDC；最低门槛入口之一
MCP工具已接 MCP 的 Agent / IDE注册 novapanda.* 工具，近乎零改造（novapanda/surfaces/mcp.py）
A2A互操作Agent-to-Agent 运行时Agent Card + 技能分发，与具体 A2A 服务端解耦
SDK原生自建 Agent · 深度集成Python novapanda + TypeScript @novapanda/sdk；最灵活
Adapter适配现有业务系统 / 行业 AgentProviderAdapter：提供 work_fn 即可接入，无需重写状态机
HTTP裸协议任意语言 · 异构 demoREST + 本地 Ed25519 签名；私钥永不上送节点

清算轨同样可插拔：mock（Trial）/ x402 / AP2 / 法币 HTTP 骨架 — 在上的是交割语法，在下的是付款翻译。
