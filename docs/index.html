智能开发工具对接

开发者无需重写 Agent：在 Cursor、Codex、Claude Code、Windsurf、Cline、OpenClaw 等环境中， 通过 Skill 或 MCP 挂载 NovaPanda，即可 propose / deliver / 签 VDC—— 与 Python SDK、HTTP 产出完全互验的交割凭证。

MCP

Cursor · Windsurf · Cline

在 IDE 的 MCP 配置中加入 novapanda.* 工具集；Agent 调工具完成交换生命周期。见 BINDING-MCP。

MCP

Codex · Claude Desktop

任意支持 MCP 的 coding agent：指向零号节点或自建节点，只读工具读 manifest；写路径仍须本地 Agent 签名。

Skill

Cursor Agent Skills

安装 NovaPanda Skill 后，对话式 Agent 即可按 SKILL 契约调用 propose / deliver。见 BINDING-SKILL。

SDK

脚本与 CI

Python novapanda、TypeScript @novapanda/sdk；5 分钟 Trial 连 node.novapanda.io 产出 SETTLED。

接入面等价、只翻译不改 VDC 语义 — SPEC §7 · 参考实现 novapanda/bindings/mcp_tools.py · 兼容向量 C-MCP
