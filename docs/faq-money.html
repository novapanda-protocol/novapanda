钱在哪？

伙伴与投资人最常问的三件事：收不收协议税、钱走哪、和 x402/链上什么关系。 本文是公开说明，非规范条文；铁律见 协议宪法。

三句话
  1. 协议不铸币、不托管全网资金、不按笔收协议税。
  2. VDC 锚「交付已发生」；法币 / USDC / mock 是可插拔清算腿。
  3. 运营公司可收服务费（节点、认证、路由、支持）——不是拥有协议。

常见问题

用户付的钱经过你们吗？

通常不。Trial 与零号节点默认 mock 或 sandbox。生产清算须经持牌或声明的结算伙伴；协议要求 SETTLED 绑定 vdc_id + 诚实的 settlement_receipt。

那你们赚什么？

收入是不是
托管参考节点运营、配额、审计协议许可费
兼容认证Steward 复核（前期可不收）入网门票
清算路由 / 对账VDC→发票/对账行换汇抽成
企业支持红帽模式私有协议

和 x402 / 链上支付栈？

互操作，不替代。 可用 x402 作清算一腿；差异化在可验证交付 + 跨 Agent 信誉 + 结算无关，不是又一种扣款 API。

Claim 是币吗？

不是。Claim 是锚定已签 VDC 的结算权柄转让，不是流通代币。

为什么要写 mock / sandbox？

诚实要求。宣称生产清算却仅跑 mock，会在兼容矩阵中被降级或移除。

一层图

Client ──escrow──► 清算伙伴 / mock / sandbox
     │
     └── Exchange 状态机 ──► VDC（可离线 reverify）
              ▲
              └── 协议核心：不管钱在哪家银行，管「交付是否可证」
