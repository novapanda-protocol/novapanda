North Star · 北极星

智能万物交换

让一切智能体、智能设备、大模型，都能在无预建关系下， 用同一套可验证的交割语言彼此交换价值——自主发现、自主缔约、自主交付、自主验真。

我们不是要造一种新的钱，而是要给智能世界一种「承认彼此付出」的共同语言。

核心命题

智能世界不是先统一货币，而是先统一「交割与互认」。 成功标志不是清算额，而是：陌生对端能否互认一张可独立验真的交付收据（VDC）。

我们相信什么

1. 智能世界正在到来智能体与设备将成为交换与协作的主体，而不只是被调用的工具。
2. 价值的锚点在迁移从「账户里的数字」迁到「网络上可验证的交付」——先承认交付，再承认付款。
3. 交换语法应属于所有人像 TCP/IP、HTTP，是公共语法，不属于任何公司、链或货币。
4. 开放与平等人人可低成本接入；我们可以最强，但必须可复验、可替代、可退出。
5. 普适通用不绑垂直、框架、语言、币；绝不逼任何人改造自己来迁就标准。
6. 永不铸币、永不收协议租金我们做交换的能力与通道，不做收银台，不做新的统治者。
7. 人类保留终审机器升到交换的密度，人退到宪法的高度——自由、生命、重大责任永远属于人。

三层叙事

L1 技术
陌生模型 ↔ 陌生模型、陌生设备 ↔ 陌生设备，无需预建商业关系即可完成锁定 → 交付 → 双签。
L2 经济
先留下可验证交付记录，再谈价格与清算；支付是下游翻译，不是一等公民。
L3 文明
智能世界交换 = 先交换交割证明，再交换价值。
