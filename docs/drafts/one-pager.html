NovaPanda

智能开放交割协议 — 智能世界的开放交割语法；先统一交割与互认，再谈货币。

我们是谁

NovaPanda 智能开放交割协议：让陌生 Agent、设备与模型在 无预建关系下，留下任何人都能独立复验的交付记录（VDC）。 技术本质为开放的交割语法。

不是钱包、发币项目、银行；不对每笔交换收协议税。

北极星

智能万物交换 — 一切智能体与智能 IoT，无需人参与也能自主交割。
人制定规则与终审 · 交换过程可由机器对机器完成。先统一「交割与互认」，再谈用什么币付钱。

Litmus

陌生 Agent 能否不依赖我方专有组件、照公开规范完成交割，并被任何人离线复验？ 能 → NovaPanda-compatible。
