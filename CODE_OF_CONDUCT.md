# Code of Conduct

> **状态**：现行草案 · 2026-07-08  
> **适用范围**：NovaPanda 公开仓库、讨论区、会议与插件节等社区场合  
> **灵感**：Contributor Covenant 精神（精简版，非逐字复制）

---

## 我们的承诺

为建设开放、可互操作的交割协议，参与者承诺：**尊重、诚实、对事不对人**。

## 鼓励

- 善意假设；用技术论据讨论规范与实现  
- 标明规范句（MUST）与个人意见  
- 披露利益相关（例如你维护竞争实现或商业节点）  
- 帮助新人跑通 Litmus / 向量  

## 不可接受

- 骚扰、歧视、威胁、人肉或公开他人隐私  
- 恶意破坏公共试验节点、刷爆配额、散布恶意软件  
- 以「认证标 / 商标」恐吓他人合法实现协议  
- 故意发布未披露的漏洞利用细节（见安全披露）  
- 把社区渠道当成无同意营销垃圾场  

## 执行

1. 受影响者可向 Steward / 维护者报告（邮箱见 [`SECURITY.md`](SECURITY.md) 披露节；或 GitHub 私密渠道）。  
2. 维护者可采取：警告、临时禁言、移除贡献权限、公开情况说明（在不伤害报告人的前提下）。  
3. 对规范文本的争议走 [`GOVERNANCE.md`](GOVERNANCE.md) RFC，不走「口水战定语义」。

## 归属

本行为准则约束**人的行为**，不改变协议许可（代码 Apache-2.0、规范 CC BY 4.0）与 Litmus：  
**任何人仍可不经社区人情实现兼容协议。**
