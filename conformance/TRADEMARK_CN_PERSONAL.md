# 中国商标申请备忘（Troodon）

> **非法律意见**。国内申请 **先中国、第 9 + 42 类**；国际以后马德里扩展。

## 个人 vs 公司（你截图里的要点）

| 维度 | 个人名义 | 公司名义 |
|------|----------|----------|
| **门槛** | 通常需 **个体工商户营业执照**；**仅身份证往往无法递交** | 需 **公司营业执照** |
| **归属** | 个人资产，与公司股权无关 | 公司资产 |
| **稳定性** | 身份证信息少变，一般不需因更名频繁变更 | 公司名/地址变更须同步变更商标 |
| **使用灵活** | 许可、转让相对自主 | 需决议、盖章；公司注销前须处理商标 |
| **使用证据** | 个体户合同/发票等 | 公司合同/发票等，相对好举证 |

**更正：** 此前说「个人 + 身份证即可马上申」——在常见实操下 **不准确**；国内多以 **个体户** 或 **公司** 作为申请主体。

---

## 对你（Troodon）的三条路

### 路径 A：先办个体工商户 → 个人/个体户名义申请（快、轻）

- 到市场监管部门登记 **个体工商户**（经营范围含软件、信息技术/技术服务相关表述，与 9/42 类商品服务项目匹配）
- 用 **个体户营业执照 + 负责人身份证** 递交商标
- **适合**：想尽快占坑、尚未注册公司；商标归你个人控制，和《行动计划》里「轻量、以后再公司化」一致

### 路径 B：先注册轻量公司 → 公司名义申请

- 需 **公司营业执照**
- **适合**：你本来就打算马上开公司（签约、CLA 受让、融资）
- 商标在公司名下；以后与 GOVERNANCE「移交基金会」要做许可/转让安排

### 路径 C：商标代理先咨询再定

- 把截图、Troodon 用途、是否有个体户/公司发给 **商标代理**
- 确认：你当前主体能否递交、9/42 类商品服务项目怎么写、与现有 Troodon 近似标是否冲突

## 建议申请内容（A/B 均适用）

| 项 | 内容 |
|----|------|
| 商标名称 | **Troodon**（纯文字或标准字体图样） |
| 类别 | **第 9 类** + **第 42 类** |
| 途径 | [sbj.cnipa.gov.cn](https://sbj.cnipa.gov.cn) 网上申请，或代理 |

## 递交前 checklist

- [ ] 近似检索（9 / 42 类）
- [ ] 申请主体就绪：**个体户执照** 或 **公司执照**
- [ ] 商标图样、申请书、商品/服务项目清单

## 递交后

- [ ] 记录 **申请号**、申请日 → 更新 [`BRAND_PREP_LOG.md`](BRAND_PREP_LOG.md)、[`docs/index.html`](../docs/index.html)
- [ ] README footnote 改为「商标申请号：xxxx」

## 与国际

- **先国内**；出海后用 **马德里**（以中国申请为基础）或国别单独申请
- 不必第一天办国际注册

## 与 GitHub / 域名

- **商标未递交前**：仓库可 public + Pages 占位，footnote「商标筹备/申请中」，不大肆宣传
- **商标受理后**：可小范围传播；主站 **https://troodon.io**
